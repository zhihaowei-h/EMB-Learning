# 线程收尸
## 所用函数
**签名**: `int pthread_join(pthread_t thread, void **retval);`
**功能**: 阻塞等待指定的线程结束，准备收线程的尸(即准备回收线程的资源)
**参数**: 
    `thread`: 指定的线程的标识
    `retval`: 获取线程退出的状态(用不到就写NULL)
**返回值**: 成功则返回0; 失败则返回错误码

## 所用函数
**签名**: `int pipe(int pfd[2]);`
**功能**: 创建管道
**参数**: 
pfd: 容量为2的int型数组
    pfd[0]: 写端
    pfd[1]: 读端

## 涉及文件
### 01pipe.c
**功能描述**: 父子进程通过文件进行通信，父进程等待子进程写入数据后再读取数据并打印出来
**完成情况**: √
### 02pipe.c 
**功能描述**: 使用匿名管道实现父子进程间通信(父进程读，子进程写)                     
**完成情况**: √
### 03pipe.c 
**功能描述**: 父进程读argv[1]文件，写进管道，然后子进程从管道中读，再写到标准输出中    
**完成情况**: √
### 04pipe.c
**功能描述**: 父进程从argv[1]文件(是一个mp3文件)中读取数据，让子进程运行一个音频播放器，来播放父进程读取到的数据，从而达到播放音乐的效果
**提示**: 如何让子进程运行一个播放器————进程替换
**前提**: 需要下载安装mplayer播放器: `sudo apt-get install mplayer`
**完成情况**: √
